# DiploSys MVP - Plataforma de Gestão de Processos Diplomáticos

---

## Visão Geral

DiploSys MVP é uma aplicação web minimalista e funcional, criada para demonstrar a ideia de uma plataforma SaaS focada na gestão digital de processos diplomáticos do CGPI do Ministério das Relações Exteriores (MRE).

O objetivo é organizar, acompanhar, analisar e automatizar processos e documentos de forma simples, eficiente e intuitiva.

---

## Funcionalidades Principais

- **Dashboard Interativo:** Estatísticas resumidas de processos e documentos.
- **Gestão de Processos:** Criar, editar, listar e acompanhar processos com status, prioridade e responsáveis.
- **Gestão de Documentos:** Upload simulado, categorização, busca e visualização.
- **Automação de Processos:** Interface para criação e gerenciamento de macros simples.
- **Análise e Relatórios:** Gráficos dinâmicos com métricas básicas.
- **Configurações:** Gerenciamento básico de usuários e configurações do sistema.

---

## Tecnologias Utilizadas

- HTML5, CSS3 (layout responsivo e moderno com cores diplomáticas)
- JavaScript (lógica de interface e manipulação de dados)
- Chart.js (gráficos dinâmicos)
- FontAwesome (ícones intuitivos)

---

## Estrutura da Aplicação

- `index.html`: Estrutura principal e navegação da aplicação.
- `style.css`: Estilos CSS para aparência e responsividade.
- `app.js`: Lógica de interface, manipulação de dados e interatividade.
- Dados mockados dentro do JavaScript simulando processos, documentos, automações e usuários.
